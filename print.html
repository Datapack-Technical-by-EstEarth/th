เกี่ยวกับ

หน้านี้สร้างขึ้นเพื่อ แนะนำ, สอน และ อธิบายเทคนิคดีๆ ในการเขียนดาต้าแพ็ค จากประสบการณ์ของผมที่ผ่านมาเพื่อให้การพัฒนากับการซ่อมบำรุงดีขึ้นและง่ายขึ้นในอนาคต ซึ่งทุกคนสามารถนำไปปรับและประยุกต์ใช้ได้ เป็นทางเลือกหนึ่งที่ไม่ได้บังคับใดๆ แต่มีจุดประสงค์เพื่อให้การปฎิบัติตาม แบบแผนทางการ นั้นง่ายขึ้น ได้รับการตรวจสอบที่ง่ายขึ้นจาก Datapack Reviewer จากปัญหาบางอย่างที่เกิดขึ้นจากประสบการณ์ที่ตรวจสอบดาต้าแพ็คมา

เครื่องมือ

นี่เป็นเครื่องมือที่ผมแนะนำมันทำให้สะดวกสบายในการเขียนดาต้าแพ็ค

โปรแกรม

1.ลง VS Code หรือ VS Code Insider หรือ VS Codium *ผมใช้ Insider

2.ติดตั้งส่วนขยาย

  • Data-pack Helper Plus สำหรับช่วยเขียนดาต้าแพ็ค

ตัวอย่างคร่าวๆ

  • language-mcfunction สำหรับไฮไลท์สีโค๊ดดาต้าแพ็คเพิ่มเติม
  • tellraw preview เอาไว้ดูตัวอย่างข้อความที่ส่งไปยังผู้เล่น

  • Comment Anchors เอาไว้ไฮไลท์คอมเม้นท์และมาร์กสีฟังก์ชั้น # ของ Data-pack Helper Plus

  • Material Icon Theme เปลี่ยนไอคอนไฟล์สวยๆ อันนี้แล้วแต่ความชอบ

ตั้งค่า

ถ้าคุณเขียนดาต้าแพ็คแล้วอยากให้โค๊ดมีลักษณะสวยๆแบบนี้ หรือ ปรับแต่งได้ตามใจชอบ สามารถตั้งค่าได้ดังนี้

1.หลังจากโหลด Font และติดตั้งแล้ว ก็ตั้งค่า Font โดยไปที่ Setting -> Text Editor -> Font -> Font Family ใส่ Fira Code

  • Font Fira Code (แล้วแต่ความชอบด้วย ที่ผมใช้ Font นี้ส่วนนึงเพราะการเปลี่ยนรูปเครื่องหมาย <= -> ดั่งในภาพ และ @ มันเท่ห์ดี)
  • Font Ligatures (เปลี่ยนรูปเครื่องหมาย <= ->) -> Edit in settings.json -> "editor.fontLigatures": true

2.ไฮไลท์สี

  • จากข้อก่อนหน้าไม่ต้องปิดหน้าต่าง settings.json
  • ให้คัดลอกนำไปใส่ได้เลย ( , จะต่อจาก editor.fontLigatures": true)
  • อยากปรับแต่งเพิ่มเติม ดูที่นี่

เคล็ดลับไฟล์ pack.mcmeta

เกี่ยวกับ

เป็นลูกเล่นที่คุณสามารถนำไปใช้ได้

ใช้งานยังไง

ไฟล์ pack.mcmeta รองรับ JSON Text Component ด้วย ซึ่งสามารถนำไปใช้กับ advancement ได้ด้วยนะ

{
	"pack": {
		"pack_format": 6,
		"description": [
			{
				"text": "verrion: 1.1.8\n",
				"color": "gold"
			},
			{
				"text": "dates: July 2020\n"
			},
			{
				"text": "author: EstEarth"
			}
		]
	}
}
